Server List

Palworld

1. First, launch Palworld and click "Join multiplayer game"
2. Then, in the field beneath the server list, enter in the server address.
3. Lastly, select the Connect button.

ARK Ascended Servers

1. First, launch Ark Survival Ascended and click "join game" via the main menu.
2. Click the "unofficial" text at the top.
3. Confirm you have the "show player servers" box checked as shown below.
4. Search the keyword "techy" and press the "enter" key twice.

ARK Survival Evolved

1. First, launch Ark Survival Evolved and click "join game" via the main menu.
2. Click the "unofficial" tab at the bottom.
3. Search the keyword "techy" and search.
